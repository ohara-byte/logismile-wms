## 6. 発注後のチェックリスト

```
□ 見積もり書受領 → 内容確認（メモリ・SSD 型番、保証期間）
□ 与信書類提出（請求書払い希望時）
□ 発注書発行 → ドスパラへ返送
□ 納期通知受領 → 設置場所準備（電源・LAN ケーブル）
□ 納品 → 動作確認（Win11 起動・LAN 疎通）
□ Tailscale セットアップへ進む（B 手順書参照）
```
